# MoreMorp
A browser extension to add more to Morp!

## Building
You must have `minify` installed, then run `build.sh`

### Firefox
Go to `about:debugging#/runtime/this-firefox` and load the temporary addon of the 'manifest.json' (MAKE SURE YOU DON'T ACCIDENTALLY PRESS `manif.json`). I suggest pressing 'Inspect' for an easy window to refresh and see logs in.
